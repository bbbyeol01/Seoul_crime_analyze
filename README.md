# 데이터 분석

주제: 서울 1만 명당 강력 범죄 건수와 유흥 업소 개수 시각화 및 상관 관계 분석

기간: 24. 10. 07 ~ 24. 10. 08 (1일)

`Python` 

`pandas` `matplotlib` 

# 목표

- 지역별 1만 명당 강력 범죄 건수를 시각화한다.
- 지역별 영업 중인 유흥 업소 개수를 시각화한다.
- 유흥 업소 개수와 강력 범죄 발생률의 상관 관계를 파악한다.
- 시각화한 그래프를 바탕으로 상대적 안전성이 높은(혹은 낮은) 지역을 도출한다.
